Chatbot as an assistant

Since the chatbot is based on the library math.js, it will be able to respond to all sorts of computation questions such as adding/multiplying, converting some units to other units, etc

You can ask the date with the date command or the more "human" commands: "what date", "what's the date"

You can ask the time for certain cities (Madrid, Hong Kong, New York): time city_name

To get a list of all supported cities: time list

Or you can ask for the time in your timezone with: time now

More info about this project: github.com/raul23/chatbots


Therapist

Talk to the program by typing in plain English, using normal upper-and lower-case letters and punctuation. Enter "quit" when done.

Info! The chatbot can also act as a "psychotherapist" (like ELIZA). I ported a Python implementation of ELIZA by Joe Strout, Jeff Epler, Jez Higgins (2005).
Warning! Don't take this "therapist" seriously. It is just a program that tries to mimic a real therapist.
